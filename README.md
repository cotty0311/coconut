## 開發階段：本機測試
### 初始化（可略）
1. 安裝 Jekyll: 首先，確保您的系統已安裝 Ruby，因為 Jekyll 依賴於 Ruby。 接著安裝 Jekyll：
```bash
gem install jekyll bundler
```
2. 初始化 Jekyll 網站: 使用以下命令初始化一個新的 Jekyll 網站：
```bash
jekyll new my-website
```
3. 進入項目並啟動伺服器: 進入您創建的 Jekyll 網站資料夾並啟動本地伺服器以查看效果：
```bash
cd my-website
bundle exec jekyll serve
```
### 啟動
- 啟動專案：Jekyll 伺服器會在 http://localhost:4000 上運行，您可以打開瀏覽器來檢視
```bash
bundle exec jekyll serve
```
